## Project Descriptions

### Eclipse Foundation
The Eclipse Foundation manages a wide range of open-source projects. [Learn more](https://github.com/eclipse).

### Eclipse Platform
The Eclipse Platform is the core framework for the Eclipse IDE. [Learn more](https://github.com/eclipse-platform).

### EGit
EGit is an Eclipse Team provider for Git. [Learn more](https://github.com/eclipse-egit).

### JDT
JDT (Java Development Tools) is part of the Eclipse IDE for Java developers. [Learn more](https://github.com/eclipse-jdt).

### PDE
PDE (Plug-in Development Environment) is a set of tools for developing Eclipse plugins. [Learn more](https://github.com/eclipse-pde).

### Equinox
Equinox is the Eclipse OSGi framework. [Learn more](https://github.com/eclipse-equinox).

### EMF
The Eclipse Modeling Framework (EMF) is a framework for building data models. [Learn more](https://github.com/eclipse-emf).

### M2E
M2E provides integration between Eclipse and Maven. [Learn more](https://github.com/eclipse-m2e).

### Oomph
Oomph simplifies the installation of Eclipse IDEs. [Learn more](https://github.com/eclipse-oomph).

### Otterdog
Otterdog is a tool for managing GitHub organizations using configuration as code. [Learn more](https://github.com/eclipse-csi/otterdog).
